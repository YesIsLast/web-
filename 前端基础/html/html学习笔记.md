# <a name="lable">常用标签</a>
## 常用标签有那些
    标签格式        有开有合，<p></p>
    头标签          <head></head>
    网页标签名      <title></title>
    标题标签        h1/h2/h3/h4/h5/h6
    正文标签        p
    超链接标签      <a> </a>    
    锚点            <a href="#锚点name">跳转到本页指定锚点</a>
                    <a name="锚点name">
    换行标签        <br />
    分割线标签      <hr>
    列表标签    
            有序列表
                    <ol>
                        <li></li>
                        <li></li>
                    </ol>
            无序列表
                    <ul>
                        <li></li>
                        <li></li>
                    </ul>
    字体加粗         <strong>******</strong>
    行内元素         <span>不设计span内样式，就无样式</span>
    图片标签         <img src="../../" height="**px" width="**px">
    平行居中标签     <center>将字段居中显示</center>
    自定义标签      
                    <dl>
                        <dt>本条列表标题(title)</dt>
                        <dd>本条列表内容</dd>
                        <dt>本条列表标题(title)</dt>
                        <dd>本条列表内容</dd>
                    </dl>
    表格标签组合
                    <table>
                        <caption>表格标题</caption>
                        <tr>
                            <th>列标题</th>
                        </tr>
                        <tr>
                            <td>列内容<td>
                        </tr>
                        
                    </table>
